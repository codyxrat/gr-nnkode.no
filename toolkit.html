Hvorfor grønn koding?

Grønn koding handler om å redusere energiforbruket og karbonavtrykket fra programvare. Ved å skrive mer optimalisert og effektiv kode kan vi bidra til en mer bærekraftig fremtid.

Bærekraftige utviklingspraksiser

  • Bruk energieffektive algoritmer
  • Optimaliser databaseforespørsler
  • Reduser HTTP-forespørsler på nettsider
  • Bruk caching for å minimere ressursbruk
  • Velg energieffektive webhoteller
  • Minifiser CSS og JavaScript

Bruk av effektive løkker i Java

For-løkker kan være mer effektive enn for-each-løkker i visse situasjoner:

public class EfficientLoop {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};
        for (int i = 0; i < numbers.length; i++) {
            System.out.println(numbers[i]);
        }
    }
}
